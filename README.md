## Description

[Dojotek](https://dojotek.id) PopVideoConvert repository. A GUI to convert video files. To allow you using the powerful FFMPEG without command line.

## Installation

Install all dependencies:

```bash
$ npm install
```

## Running the app

To run the PopVoiceOver GUI:

```bash
$ npm start
```
